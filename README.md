# Battleship

Данный проект является реализацией игры "Морской Бой" с возможностью играть по сети против других игроков или против компьютера.

## Особенности
- Игра по сети с другим игроком.
- Возможность игры против компьютера.
- Размещение кораблей вручную или случайным образом.
- Индикация попаданий, промахов и потопленных кораблей.
- Удобный графический интерфейс с отображением досок игроков.

## Системные требования
- Java 8 или выше.
- Любая операционная система, поддерживающая Java.

## Установка и запуск
1. Скачайте готовые `.jar` файлы игры с релиза
2. Для запуска сервера используйте команду: ```java -jar "путь к вашему файлу"\battleship-server.jar```
3. Для запуска клиента используйте команду: ```java -jar "путь к вашему файлу"\battleship-client.jar```
